# WSL NixOS 个人配置

基于 **WSL 2 + NixOS** 的个人开发环境配置，专注于可复现性、快速初始化和稳定回滚。

## 功能特点

- ✅ **多语言开发环境**
  - 内置 Rust、Python 等常用语言工具链
  - 统一由 Nix 管理，避免环境污染

- ✅ **开箱即用的编辑器体验**
  - 预配置 Neovim
  - 集成常用 LSP（如 rust-analyzer、pyright）
  - 支持格式化、补全与诊断

- ✅ **原子化配置与安全回滚**
  - 所有系统与用户配置均由 Nix 管理
  - 配置变更可随时回滚，降低试错成本

## 适用环境

- Windows 10 / 11
- WSL 2
- NixOS (WSL 发行版)

## 设计目标

- 最小但实用
- 易于扩展
- 长期可维护
